# Vellichor Product Introduction

Landing page for Vellichor — an AI-powered platform that captures and preserves organizational tacit knowledge.

## Overview

Vellichor is your company's superintelligence. We help organizations capture the intuition, not just the information, preserving your team's hard-won wisdom before it walks out the door.

## Features

- **Multimodal Capture** - Turn whispers of wisdom into enterprise assets through voice, video, and document analysis
- **Expert Agent** - Instant access to collective organizational intuition whenever and wherever you need it
- **Scale Intuition** - Convert years of accumulated wisdom into multiplied strategic value across your organization
- **Local Intelligence** - On-premise deployment keeps sensitive insights within your walls
- **Tacit Backbone** - Seamlessly enhance your existing tools with embedded institutional wisdom

## Tech Stack

- HTML5
- CSS3 (with CSS Custom Properties)
- Vanilla JavaScript
- Google Fonts (Inter, Open Sans)

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/dave-song/vellichor_product_intro.git
   ```

2. Open `index.html` in your browser, or use a local development server:

   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve
   ```

3. Navigate to `http://localhost:8000` in your browser.
